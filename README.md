# Personal Portfolio Website

This repository contains a fully static personal portfolio website for a software engineer turned cloud engineer. It is built with plain HTML, CSS, and vanilla JavaScript only, with no build tools or framework dependencies.

## Run locally

No setup is required:

1. Open `index.html` directly in your browser.

## Deployment

This project is configured for GitHub Pages as static root files:

- Source: `main` branch
- Folder: `/ (root)`

Once pushed to GitHub, enable GitHub Pages in repository settings with that source configuration.
